Section A: Multiple Choice Questions (MCQs)

1. Which of the following is a primary function of an operating system?

Section B: Short Answer Questions

1. Define system software.

Answer: System software is software that controls computer hardware and provides an environment for running application software.

2. List two main types of user interfaces provided by operating systems.

Answer: Command-line interface (CLI) and Graphical User Interface (GUI).

3. What is the role of an operating system in program–hardware interaction?

Answer: The OS ensures that hardware executes the instructions requested by software correctly and efficiently.

4. Explain what is meant by a process in an operating system.

Answer: A process is a program that has begun execution and is currently managed by the operating system.

5. Name two tasks involved in memory management.

Answer: Memory protection and memory usage optimization (including virtual memory management).

6. What is the purpose of a device driver?

Answer: A device driver allows the operating system and software to communicate with and control hardware devices.

7. List three features of file management in an OS.

Answer: File naming conventions, directory/folder structures, and access control mechanisms.

8. Explain the function of a disk defragmenter.

Answer: A disk defragmenter reorganizes fragmented files on a disk so that each file occupies contiguous sectors, improving performance.

9. What is the purpose of backup software?

Answer: Backup software creates copies of data according to a schedule to prevent data loss.

10. Explain the advantage of a Dynamic Link Library (DLL).

Answer: DLLs allow multiple programs to share routines in memory, reducing storage and memory usage while enabling updates to routines without modifying each program.

11. What is the difference between an interpreter and a compiler?

Answer: An interpreter executes source code line by line and detects errors immediately, while a compiler translates the entire source code into object code for later execution.

12. Give one advantage of using an IDE for programming.

Answer: An IDE can provide features like syntax highlighting, debugging tools, and auto-completion to help programmers write and correct code efficiently.

13. What is meant by “dynamic syntax check” in an IDE?

Answer: Dynamic syntax check automatically detects and highlights syntax errors as the programmer types code.

14. Explain why operating systems are required to perform error detection and recovery.

Answer: Operating systems detect errors to prevent incorrect execution of programs or hardware failures, and provide recovery mechanisms to avoid data loss.

15. Define utility software and give two examples.

Answer: Utility software is a program that performs maintenance or management tasks for the computer. Examples include disk repair tools and virus checkers.

Section C: Exam Style Questions

1. Describe the main functions of an operating system and explain why it is essential for modern computers.

Answer: The OS manages hardware resources, provides a user interface, controls processes, memory, devices, files, security, and error recovery. It is essential because it enables efficient use of hardware and smooth execution of software.

2. Explain the difference between system software and application software with examples.

Answer: System software controls hardware and provides an environment for applications (e.g., operating systems, device drivers). Application software performs tasks for users (e.g., word processors, games).

3. Discuss the concept of a process and explain process scheduling in modern operating systems.

Answer: A process is a running program. Process scheduling determines the order in which multiple processes access CPU and other resources, ensuring fairness, efficiency, and avoiding conflicts.

4. Describe memory management in operating systems, including memory protection and virtual memory.

Answer: Memory management allocates memory to processes efficiently. Memory protection prevents one process from accessing another's memory. Virtual memory allows a computer to use disk space as an extension of RAM, supporting larger programs.

5. Explain the roles of device management and file management in operating systems.

Answer: Device management installs drivers and controls device usage. File management provides naming conventions, directory structures, and access control to store, organize, and retrieve files.

6. Discuss the importance of security management and error detection in an OS.

Answer: Security management prevents intrusion, ensures privacy, and provides recovery from data loss. Error detection allows the OS to handle program or device errors and shut down safely if needed.

7. Explain what utility software is and describe the functions of disk formatter, defragmenter, and backup utilities.

Answer: Utility software performs maintenance tasks. Disk formatter sets up the file system, defragmenter reorganizes fragmented files, and backup software copies data for recovery.

8. Describe the concept of program libraries and the advantages of using DLLs.

Answer: Program libraries are collections of reusable subroutines. DLLs allow multiple programs to share routines, saving storage and memory, and enable automatic updates of routines.

9. Explain the difference between compilers and interpreters and their advantages for programmers.

Answer: Interpreters execute code line by line, helping detect errors during development. Compilers translate entire code to object code, providing faster execution and enabling distribution without exposing source code.

10. Explain how Java achieves platform independence using the Java Virtual Machine (JVM).

Answer: Java compiles programs into bytecode, which is interpreted by the JVM on any platform. This allows the same Java program to run on different hardware and OS without modification.

11. Describe the features found in an Integrated Development Environment (IDE) that assist programmers.

Answer: IDE features include prettyprinting, context-sensitive prompts, dynamic syntax checks, code collapsing, and debugging tools, all helping programmers write and correct code efficiently.

12. Explain the process of debugging using breakpoints in an IDE.

Answer: Breakpoints pause program execution at a selected point, allowing step-by-step execution to examine variable values and program flow, making it easier to find and correct errors.

13. Discuss why file compression utilities are useful in operating systems.

Answer: File compression reduces storage space requirements and makes file transfer faster, especially useful when sending files over the internet.

14. Explain the importance of virus checkers and the need for regular updates.

Answer: Virus checkers prevent malware infections. Regular updates are needed to detect new viruses, as there is a delay between virus emergence and definition updates.

15. Explain why operating system tasks can be categorized as assisting users vs running the system.

Answer: User-assisting tasks provide interfaces and utilities for user convenience, while system-running tasks manage hardware, processes, memory, and files to ensure smooth operation.
