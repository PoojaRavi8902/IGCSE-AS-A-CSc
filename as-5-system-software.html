Translators

Programs written in high-level or assembly languages cannot be directly executed by the CPU. Translators bridge this gap by converting source code into machine code.

Key Term: Translator – A program that converts source code into machine-readable machine code.

Compiler

A compiler translates the entire program into machine code at once, creating a standalone executable file. Once compiled, the program runs very quickly.

Example: GCC compiles C programs into efficient executables.
Exam Tip: Remember that compiled programs take longer to prepare but execute faster.

Interpreter

An interpreter translates source code line-by-line, executing it immediately. While slower, it allows for real-time debugging.

Example: Python uses an interpreter, making it excellent for rapid prototyping.

Assembler

An assembler converts assembly language into machine code. Assembly language is low-level but easier than binary for humans to understand.

Operating Systems

An Operating System (OS) is the core software that manages hardware and software resources, enabling user interaction with the system.

Core Functions

  • Memory Management: Allocates and deallocates memory between processes.
  • Processor Scheduling: Determines which processes use the CPU and when.
  • File Management: Controls data storage, retrieval, and permissions.
  • Device Management: Manages printers, keyboards, monitors, etc.
  • Security: Provides authentication, encryption, and access control.
  • User Interface: Supplies CLI or GUI for human-computer interaction.
Task: Compare how Linux, Windows, and macOS handle security updates differently.
Exam Tip: Be ready to give examples of OS functions in real life (e.g., Task Manager = process scheduling).

Utility Programs

Utility software performs specialized tasks to maintain, optimize, or protect the system.

  • Backup: Creates copies of data for recovery after loss.
  • Antivirus: Protects against viruses and malware.
  • Compression: Reduces file sizes for storage or transfer.
  • Disk Management: Formats, partitions, and monitors disk health.
Case Study: Hospitals use RAID backup utilities to ensure patient records are never lost.

Summary & Revision Checklist

  • Translators: compiler, interpreter, assembler.
  • OS: manages memory, CPU, files, devices, security, and UI.
  • Utilities: backup, antivirus, compression, disk management.
