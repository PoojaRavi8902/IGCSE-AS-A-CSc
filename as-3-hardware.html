This chapter covers hardware components of computer systems, including storage, input and output devices, memory components, and embedded systems. Understanding these is essential for analysing computer architecture and system design.

3.01 Overview of Computer System Hardware Functionality

A computer system supports three major operational areas:

  • Processing of data (carried out by the CPU – discussed in Chapter 5)
  • Storage of data (primary and secondary storage)
  • Input and output of data (I/O subsystem)

3.02 Embedded Systems

An embedded system is a special-purpose computer within a larger device, often built on a microcontroller (processor + memory + I/O on one chip).

Examples: household appliances, cars, smartphones, industrial machines.

Advantages: mass production lowers costs, designed for single function, efficient use of resources. Risks: limited programmability, security concerns when connected to IoT.

Discussion Point: How might useful information from an embedded system in a washing machine be communicated to its owner via the internet?

3.03 Memory Components

RAM (Random Access Memory)

  • Volatile: data lost when power is off.
  • Dynamic RAM (DRAM): cheaper, higher capacity, slower (needs refreshing).
  • Static RAM (SRAM): faster, more expensive, used in cache.

ROM (Read-Only Memory)

  • Non-volatile: retains data when power is off.
  • Stores bootstrap program and firmware.
  • Types: ROM, PROM, EPROM, EEPROM.

Buffers

Used to manage data transfer between devices with different speeds. Acts as a queue to smooth data flow.

3.04 Secondary Storage Devices

Storage media categories:

  • Magnetic media: Hard disks, magnetic tapes (long-term backup).
  • Optical media: CDs, DVDs, Blu-ray (laser technology).
  • Solid-state media: Flash memory (USB sticks, SSDs).

3.05 Output Devices

  • Screens: CRT, LCD, LED, OLED, VR headsets.
  • Printers: Inkjet, Laser (high-quality hardcopy).
  • Plotters: For vector graphics, technical drawings.
  • 3D Printers: Additive manufacturing.
  • Speakers: Audio output.

3.06 Input Devices

  • Keyboards and keypads
  • Pointing devices (mouse, trackpad, touchscreen)
  • Game controllers
  • Scanners and webcams
  • Microphones (with voice recognition)
  • Reading from storage devices
  • Transmission via network link
Discussion Point: Investigate which touch screen technology (resistive or capacitive) is used in your phone or laptop and discuss its advantages.

3.07 Input and Output of Sound

Sound input requires a microphone (converts sound to electrical signal → ADC → digital). Sound output requires a speaker (digital → DAC → electrical → sound).

Applications: IP telephony, video conferencing, voice recognition, voice synthesis, music recording/playback.
